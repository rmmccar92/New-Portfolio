Featured Project

Confused Gamer

Using an video game database API known as Giant Bomb my group and I created the Confused Gamer application. In this application we aimed to consolidate a users' choice when considering the vast array of gaming titles available. Using the API we displayed information about various games by console and allowed users to pick and choose games that they might be interested in.
